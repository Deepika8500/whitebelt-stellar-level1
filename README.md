# WhiteBelt - Stellar Level 1

## Project Description
Completed Stellar WhiteBelt Level 1 tasks:
1. Created a new Stellar Testnet Account
2. Funded the account with Friendbot - 10000 XLM
3. Verified the transaction on Stellar Lab

 # Setup Instructions
1. Go to https://whitebelt.stellar.org
2. Click "Create Account" to generate Public and Secret Key
3. Click "Fund with Friendbot" to get 10000 XLM testnet
4. Verify balance and transaction at https://laboratory.stellar.org
